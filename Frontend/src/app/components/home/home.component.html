¿Que es EnJob?

Enjob es una aplicación web que busca mejorar la satisfacción laboral de los empleados utilizando herramientas y estrategias que permitan lograr este objetivo. Entre las funcionalidades que ofrece EnJob, ofrece un tablero Kanban para la visualización de las tareas, un chat interno entre los empleados y el encargado de los recursos humanos, de igual forma, ofrece un sistema de notificaciones para presentar alguna queja o mensaje a el departamento de recursos humanos, un sistema de notificaciones para presentar quejas o un mensaje al departamento de RRHH, EnJob tambien ofrece un sistema de logros por medio de puntos para favorecer la incentivación de los empleados y el control estadístico de los empleados por parte del departamento de RRHH para la visualización eficaz del desempeño de los empleados a traves del tiempo.
